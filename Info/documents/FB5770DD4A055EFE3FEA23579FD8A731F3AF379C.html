MultiChain, the private blockchain platform launched by Coin Sciences Ltd, has entered beta phase with the release of MultiChain 1.0 for Linux and Windows.
 
The platform has also revealed that 15 new industry partners are joining its Platform Partner Program – a consultancy group originally backed by financial services giant Accenture.
 
Notably, the new members of the program include three multinationals – Boston Consulting Group, PwC and Worldline – as well as 11 smaller companies.
 
The MultiChain platform was set up with the aim of helping organizations more easily build applications using blockchains and distributed ledgers.

MultiChain 2.0 preview: Smart Filters
 
We're delighted to deliver the fifth preview release of MultiChain 2.0, with a huge new feature: Smart Filters.
 
A Smart Filter is a piece of code which is embedded in the blockchain, and which allows custom rules to be defined regarding the validity of transactions or data.
 
Smart Filters are written in JavaScript and run within a deterministic version of Google’s V8 JavaScript engine, which is embedded directly within MultiChain. This is the same high performance JavaScript compiler used in Google Chrome, Node.js and many other platforms.
 
If you've been waiting for MultiChain to support "smart contracts", Smart Filters are your answer. Just like smart contracts, Smart Filters allow custom blockchain code to define complex rules for permissioning, time limits, third party approvals, data validation and more. But unlike the smart contract model, Smart Filters let you add just the extra rules that you need, and don't force you to express every blockchain operation as the execution of code.
