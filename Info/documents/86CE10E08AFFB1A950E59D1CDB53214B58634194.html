InterPlanetary File System (IPFS) is a protocol designed to create a permanent and decentralized method of storing and sharing files.[2] It is a content-addressable, peer-to-peer hypermedia distribution protocol. Nodes in the IPFS network form a distributed file system. IPFS is an open-source project developed since 2014 by Protocol Labs with help from the open-source community.[3] It was initially designed by Juan Benet.[4]
