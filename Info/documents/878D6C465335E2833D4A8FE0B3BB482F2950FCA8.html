Roadmap is long
 
From WP:
OTN.x: Guaranteed interaction between blockchains
We want to implement atomic, guaranteed cross-chain interactions between systems outside the OTN network. For example, it would be possible to exchange BTC for ETH with guarantees inside OTN. This exchange could be conveniently made from the OTN wallet or from external interfaces through the public API.
 
We intend to implement support for the major cryptocurrencies for such integration in upcoming iterations. To connect an external system to the network, it will only be necessary to implement the interface for it and integrate the module in an OTN.x node. This interaction will be standardized. In other words, the OTN.x technology will allow the OTN ecosystem to perform validation functions in third-party networks through a network of distributed monitors. This idea can be extended to any processes that can use decentralized validation with sufficient guarantees. This principle applies to external Market Data providers or document registries. We plan to create a universal interface for conducting operations with such systems with a sufficient level of safeguards. This interface may become a new standard for exchanging assets between any systems with the security guarantees of the OTN ecosystem.
